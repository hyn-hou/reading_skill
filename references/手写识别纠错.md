# 手写体样本与识别纠错库

本文件收录小学生常见书写习惯样本和OCR常见误识别模式与纠正规则。

---

## 一、小学生书写特点

### 1-2年级特点
| 特点类型 | 具体表现 |
|----------|----------|
| 笔画不规范 | 横不平、竖不直、弯钩不到位 |
| 字体大小不一 | 大字小字混杂 |
| 位置偏移 | 写字歪斜、越格 |
| 涂改较多 | 用橡皮擦、用笔划掉 |
| 拼音夹杂 | 不会写的字用拼音代替 |
| 方向反转 | 左右偏旁互换（如"情"写成"忄青"） |

### 3-4年级特点
| 特点类型 | 具体表现 |
|----------|----------|
| 书写速度提升 | 字迹相对工整 |
| 连笔出现 | 部分笔画连写 |
| 涂改减少 | 整体较为整洁 |
| 结构改善 | 字形结构趋于稳定 |

### 5-6年级特点
| 特点类型 | 具体表现 |
|----------|----------|
| 字体成型 | 基本形成个人书写风格 |
| 速度较快 | 整体较为流畅 |
| 偶有潦草 | 部分学生书写较快时字迹潦草 |

## 二、OCR误识别模式

### 字形混淆

| 正确字 | 易误识为 | 原因 |
|--------|----------|------|
| 己 | 已、巳 | 起笔不同 |
| 土 | 土、士 | 横的长短 |
| 大 | 太、犬、大 | 是否有捺点 |
| 人 | 入、八 | 撇捺长短 |
| 刀 | 力、刀 | 弯钩方向 |
| 了 | 阝、子 | 钩的方向 |
| 日 | 目、白 | 内部横线 |
| 目 | 日、月 | 形状相似 |
| 己 | 已、巳 | 竖弯钩 |
| 午 | 牛、干 | 横的长短 |

### 相近字混淆

| 正确字 | 易误识为 | 区分特征 |
|--------|----------|----------|
| 情 | 清、请、晴 | 竖心旁/月/日 |
| 青 | 清、情、晴、蜻 | 需要上下文 |
| 公 | 分 | 人字头 |
| 采 | 菜 | 爪字头/艹 |
| 师 | 帅 | 长撇位置 |
| 弯 | 湾 | 丝/氵 |
| 蜜 | 密 | 宀/山 |
| 象 | 像 | 人/亻 |

### 常见错字形态

| 正确 | 常见错误 | 说明 |
|------|----------|------|
| 鸟 | 乌 | 少一点 |
| 兔 | 免 | 多一点 |
| 练 | 炼 | 绞丝旁/火 |
| 琴 | 瑟 | 王/玨 |
| 纸 | 抵 | 绞丝旁/提手旁 |

## 三、识别纠错规则

### 错别字自动纠正

```
常见错字 → 正确字 对照表：

冷 → 冷（"冷"易写成"冫令"）
满 → 满（"艹"易写成"忩"）
很 → 很（"彳"易写成"亻"）
着 → 着（下面容易写成"目"）
真 → 真（中间容易少横）
真 → 直（区别）
```

### 语义纠错判断

当OCR识别存在歧义时，结合语义上下文判断：

```
示例1：
识别结果："小鱼在水里游来游去"（可能有"鱼"和"渔"混淆）
判断依据：前文提到"小鱼"→ 确认为"鱼"

示例2：
识别结果："小鸡在草地捉虫"（可能有"捉"和"抓"混淆）
判断依据：语义通顺，两者均可接受 → 标记为合理

示例3：
识别结果："小鸟在天上飞"（可能有"上"和"下"混淆）
判断依据：结合生活常识 → 确认为"上"
```

### 涂改痕迹处理

```
1. 优先识别最后书写的内容
2. 若无法确定，标记为"[涂改]"并提示教师确认
3. 常见涂改位置不作为识别依据
```

### 拼音处理规则

```
1. 识别拼音字母
2. 尝试匹配对应汉字
3. 若匹配失败，保留拼音并标记"[拼音]"
4. 提示：可结合上下文或查看原图确认
```

## 四、置信度标记规则

### 低置信度标记情况

| 情况 | 标记方式 | 处理建议 |
|------|----------|----------|
| 字形完全无法辨认 | [???] | 教师确认 |
| 多个候选字难以区分 | [候选字/字] | 教师选择 |
| 涂改无法确定 | [涂改] | 教师确认 |
| 拼音无法转换 | [拼音:xxx] | 教师输入 |
| 模糊不清 | [模糊] | 教师确认 |

### 置信度阈值

- **高置信度** (>95%)：自动采用
- **中置信度** (80%-95%)：采用并标绿
- **低置信度** (<80%)：标记并请教师确认

## 五、书写样本参考

### 低年级常见书写样式

```
典型特征：
- "的"字钩较长
- "心"字卧钩较平
- "小"字竖钩较短
- "山"字中间竖较短
```

### 常见偏旁写法

| 偏旁 | 正确写法 | 常见错误 |
|------|----------|----------|
| 讠 | 点、横折提 | 横写成长横 |
| 阝 | 横撇弯钩、竖 | 弯钩写成竖弯钩 |
| 忄 | 左点、右点、竖 | 竖写成垂露 |
| 氵 | 点、点、提 | 三点连成一线 |
| 艹 | 点、点、短竖 | 两草头混淆 |

## 六、图像预处理建议

### 质量检测
1. 检查图片是否清晰
2. 检查光照是否均匀
3. 检查是否有阴影或反光
4. 检查文字方向是否正确

### 预处理操作
1. **灰度化**：转换为灰度图像
2. **二值化**：提高文字对比度
3. **去噪**：去除背景噪点
4. **倾斜校正**：矫正歪斜的文字
5. **区域分割**：分离文字区域

### 典型问题处理

| 问题 | 预处理方法 |
|------|------------|
| 光线过暗 | 提高亮度/对比度 |
| 光线过亮 | 降低对比度 |
| 文字歪斜 | 倾斜校正 |
| 背景复杂 | 背景消除 |
| 字迹模糊 | 锐化处理 |
