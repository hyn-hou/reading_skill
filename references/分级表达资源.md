# 分级表达资源库

本文件收录各年级常用评价话术、过渡语、句模，保证语言一致性和适龄性。

## 第一学段（1-2年级）表达资源

### 鼓励性语句
```
你真棒！
写得很清楚！
已经说对了一部分，继续加油！
这个意思表达得很好！
你的答案越来越完整了！
很用心，继续努力！
```

### 具体肯定语
```
你找到了XX这个词，说明你认真读了课文。
你把XX（主要内容）说出来了，真不错。
你的答案比上次更完整了！
```

### 引导补充语
```
再想一想，还可以写什么？
如果把XX也加进去，答案就更完整了。
想一想，故事里还提到了什么？
还可以说一说XX。
```

### 温和指正语
```
这里可以换个说法试试。
再读一读题目，看看有没有漏掉什么。
老师知道你已经努力了，试试这样表达……
```

## 第二学段（3-4年级）表达资源

### 整体评价语
```
你的回答抓住了主要内容。
对题意理解正确，表达也清楚。
你的答案覆盖了大部分要点。
这道题需要再补充一两个方面会更完整。
```

### 逐点肯定语
```
✓ 关于XX这一点，你说得很清楚……
✓ 你提到的XX，说明你理解了这个内容。
✓ 你的回答涉及了XX这一方面，很好。
```

### 引导深入语
```
关于这一点，能不能说得更具体一些？
你能联系上文说一说原因吗？
如果再加上XX（补充角度），会更全面。
```

### 改进建议语
```
你的答案缺少了对XX的说明。
关于XX这一点，可以这样补充……
如果要回答得更完整，建议从XX和XX两个角度来考虑。
可以试着用"因为……所以……"的句式来说明原因。
```

### 关联课文语
```
联系课文想一想……
还记得课文是怎么说的吗？
文中有一句话可以帮你回答这个问题……
```

## 第三学段（5-6年级）表达资源

### 高阶评价语
```
你的回答紧扣题意，要点清晰。
你能够从多个角度来分析这个问题，很好。
你对文章内容的理解比较深入。
表达有条理，能看出你是认真思考过的。
```

### 写作手法点评语
```
你注意到了作者运用的XX（写作手法），这说明你读得很仔细。
你提到的这句话用了XX的修辞手法，生动地表达了……
从你的回答可以看出，你理解了文章的XX（写作方法）。
```

### 深度理解引导语
```
能不能深入分析一下原因？
你觉得作者为什么要这样写？
这篇文章的主题是什么？你是怎么理解的？
如果是你来写，你会怎样表达？
```

### 综合建议语
```
你的答案框架是对的，可以从XX角度再深入一点。
建议先明确观点，再结合文中内容说明理由。
可以试着用"首先……其次……最后……"的结构来组织答案。
关于这一点，最好能联系文章中心思想来回答。
```

### 情感体悟引导语
```
读完这部分，你有什么感受？
你觉得作者想通过这篇文章表达什么？
文中的XX让你产生了什么情感共鸣？
```

## 通用过渡语

### 表示理解的过渡
- 我理解你的意思是……
- 你的想法是对的，但可以这样说……
- 你说得很对，不过还差一点……

### 表示补充的过渡
- 除此之外，还可以说……
- 还有一个方面是……
- 从另一个角度看……

### 表示总结的过渡
- 总的来说……
- 综合以上……
- 概括起来……

### 表示递进的过渡
- 不仅如此……
- 更重要的是……
- 进一层来说……

## 分年级句模

### 1-2年级答题句模
```
格式：谁+做了什么/是什么
例：小白兔去采蘑菇。
例：太阳是红红的、暖暖的。
```

### 3-4年级答题句模
```
格式：因为……所以……（因果关系）
格式：先……然后……最后……（叙事顺序）
格式：虽然……但是……（转折关系）
```

### 5-6年级答题句模
```
格式：这篇文章主要写了……（概括主要内容）
格式：从XX可以看出……（分析写作手法）
格式：联系上下文，我理解了……（词句理解）
格式：我的观点是……理由如下……（表达观点）
```
